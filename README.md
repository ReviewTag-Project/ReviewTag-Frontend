# ReviewTag - 콘텐츠 리뷰 및 퀴즈 플랫폼

React와 Vite를 기반으로 구축된 콘텐츠(영화, 드라마 등) 리뷰 및 퀴즈 커뮤니티 플랫폼입니다.

## 주요 기능

### 콘텐츠 관리
- 콘텐츠 검색 및 상세 조회
- 장르별 콘텐츠 목록 조회
- 콘텐츠 리뷰 작성 및 관리
- 콘텐츠별 퀴즈 생성 및 참여

### 회원 기능
- 회원가입 및 로그인
- 마이페이지 (내 정보, 내 퀴즈, 내 게시글, 내 콘텐츠, 내 즐겨찾기, 내 리뷰)
- 프로필 조회 (타인 프로필, 리뷰, 즐겨찾기 조회)
- 회원 정보 수정 및 비밀번호 변경

### 포인트 시스템
- 출석 체크 (출석 달력)
- 룰렛 게임으로 포인트 획득
- 포인트 스토어 (굿즈 구매)
- 인벤토리 관리
- 포인트 기부
- 포인트 랭킹
- 일일 퀘스트

### 커뮤니티
- 게시판 (콘텐츠별 게시글 목록)
- 게시글 작성, 수정, 삭제
- 리뷰 작성 및 수정

### 퀴즈
- 콘텐츠별 퀴즈 생성
- 퀴즈 게임 참여
- 내가 만든 퀴즈 관리

### 랭킹
- 퀴즈 랭킹
- 리뷰 랭킹
- 콘텐츠 랭킹
- 신규 랭킹

### 관리자 기능
- 회원 관리
- 게시판 관리
- 퀴즈 관리
- 일일 퀴즈 설정
- 포인트 관리
- 인벤토리 관리
- 스토어 관리
- 리뷰 신고 관리

## 기술 스택

### 프론트엔드
- **React 19.2.0** - UI 라이브러리
- **Vite 7.2.4** - 빌드 도구
- **React Router DOM 6.30.2** - 라우팅
- **Jotai 2.15.2** - 상태 관리
- **Axios 1.13.2** - HTTP 클라이언트
- **Bootstrap 5.3.8** - UI 프레임워크

### 주요 라이브러리
- **react-quill-new** - 리치 텍스트 에디터
- **react-icons** - 아이콘
- **react-toastify** - 토스트 알림
- **sweetalert2** - 알림 모달
- **react-calendar** - 캘린더 컴포넌트
- **react-slick** - 캐러셀 슬라이더
- **moment** - 날짜 처리
- **lodash** - 유틸리티 함수

## 프로젝트 구조

```
src/
├── components/          # 컴포넌트
│   ├── admin/          # 관리자 페이지 컴포넌트
│   ├── board/          # 게시판 컴포넌트
│   ├── Contents/       # 콘텐츠 관련 컴포넌트
│   ├── error/          # 에러 페이지
│   ├── guard/          # 인증 가드 (Private, Admin)
│   ├── member/         # 회원 관련 컴포넌트
│   ├── point/          # 포인트 시스템 컴포넌트
│   ├── quiz/           # 퀴즈 관련 컴포넌트
│   ├── ranking/        # 랭킹 페이지 컴포넌트
│   └── review/         # 리뷰 관련 컴포넌트
├── utils/              # 유틸리티
│   ├── axios/          # Axios 설정
│   ├── jotai/          # Jotai 상태 관리
│   └── localStorage/   # 로컬 스토리지 관리
├── App.jsx             # 메인 App 컴포넌트
└── main.jsx            # 애플리케이션 진입점
```

## 설치 및 실행

### 요구사항
- Node.js (권장 버전: 18 이상)
- npm 또는 yarn

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버는 기본적으로 `http://localhost:5173`에서 실행됩니다.

### 빌드

```bash
npm run build
```

### 빌드 미리보기

```bash
npm run preview
```

### 린팅

```bash
npm run lint
```

## 환경 변수

프로젝트는 Vite의 환경 변수를 사용합니다. `.env` 파일을 생성하여 다음 변수를 설정하세요:

```env
VITE_BASE_URL=/
```

## 주요 페이지 구조

- `/` - 홈 페이지
- `/contents/detail/:contentsId` - 콘텐츠 상세 페이지
- `/review/write/:contentsId` - 리뷰 작성
- `/member/mypage` - 마이페이지
- `/point/main` - 포인트 메인 (인증 필요)
- `/board/list` - 게시판 목록
- `/quiz` - 퀴즈 랜딩
- `/ranking` - 랭킹 페이지
- `/admin` - 관리자 페이지 (관리자 권한 필요)

## 인증 및 권한

- **Private**: 로그인한 사용자만 접근 가능
- **Admin**: 관리자 권한이 있는 사용자만 접근 가능
